# social-post 輸出模板

## 1. 正式選項題模板

```md
題目：<讀者 / 痛點 / 收穫 / 場景 / 公式>
推薦：<先說明第 1 個為推薦項>
選項：
- A. <候選 1>
- B. <候選 2>
- C. <候選 3，可省略>

若沒有正式互動式選項工具，維持同樣候選內容，改用文字讓使用者回 A / B / C。
```

## 4.2 文章評估模板

```md
## 文章評估

- 標題穿透力
  - 3 秒內看得懂這篇給什麼：
  - 有沒有具體性 / 權威性 / 認知衝擊：

- 開場鉤子強度
  - 前三段有沒有講清楚主題、價值、為什麼要看：
  - 有沒有過度鋪陳：

- 串接與切角
  - 前言接主體是否順：
  - 每個 `▋` 是否有獨特角度：

- 結構完整性
  - 每段是否有開場 / 展開 / 收尾：
  - 案例是否有具體細節：

- 收斂程度
  - 是否刪掉冗句與重複：
  - 是否保留必要細節與解析度：
  - 是否比原版節奏更順：
  - 是否精簡後仍保留人味：

- 收尾完整性
  - 是否有重點回顧：
  - 是否有明確 CTA 或下一步：

- 人味與作者存在感
  - 是否有真實場景：
  - 是否有主觀判斷句：
  - 是否有自然情緒 / 自嘲：
  - 是否仍有太像 AI 或官方說明的段落：
```
