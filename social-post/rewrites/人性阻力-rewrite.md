# 人性阻力

我後來理解到一個很實用的道理。

很多東西推不動，

真的不是因為它不夠好。

而是因為它讓人動起來很麻煩。

這個麻煩，不一定是表面的麻煩。

有時候不是貴。
不是難。
不是沒價值。

而是它會讓人不舒服。

這種不舒服，如果用一句話講：

**阻力。**

​▋ ​大部分人都太愛講吸引力，太少去看阻力

做產品的人很容易有一個習慣。

一直講自己有多好。

功能多強。
價值多大。
這個東西多值得買。

但很多時候，真正讓人沒下單的，

根本不是他不知道你好。

而是他腦袋裡還有一堆沒被處理掉的小阻力。

像賣沙發這件事就很典型。

客人明明很喜歡。
現場也很想買。

結果回家之後就消失了。

後來才發現，問題根本不是沙發。

而是老沙發怎麼處理。

這種感覺我很有感。

因為很多事情卡住，往往不是主體問題，

而是旁邊那一段沒有被接好。

你以為大家在評估價值，

其實很多人卡的是：

1. 我後面要多做什麼？
2. 這件事會不會很麻煩？
3. 我是不是還要重新適應？
4. 做了之後會不會更亂？

這些東西很少會被直接講出來。

但它才是真正讓人停住的地方。

​▋ ​我最後留下來的一個好用框架，是把阻力拆成四種

我自己讀完，最有收穫的不是案例。

是它讓我多了一個很實用的分類方式。

它把人性的阻力拆成四種：

1. 慣性
2. 惰性
3. 情感
4. 抵觸

這四個看起來很簡單，

但你一旦開始用這四個角度去看事情，很多原本模糊的卡點會突然變清楚。

### 1. 慣性

不是這個新東西不好。

是它離現在太遠。

人對熟悉的東西，本來就比較有安全感。

就算原本那套沒多好，

只要夠熟，

很多人就寧可繼續待在那裡。

這件事在工作裡超常見。

很多流程大家不是不知道爛。

只是你一旦叫他改，他第一個反應不是「好像更好」，

而是「我又要重新學一遍」。

### 2. 惰性

這個不是在罵人懶。

比較像是：

做這件事到底要不要多花力氣。

如果改變很麻煩，
步驟很多，
決策很重，
還要多花錢，

那很多人就會停住。

不是因為不認同。

是因為懶得動。

### 3. 情感

這個最容易被忽略。

因為很多時候，人嘴巴上講的是功能，

心裡真正卡的是「感受」。

裡面提到像雞蛋粉那個例子。

本來廠商覺得自己做得很好，

連雞蛋粉都幫你加好了，你只要加水就能烤。

結果賣不好。

後來把雞蛋粉拿掉，叫使用者自己打一顆蛋，

銷量反而起來。

為什麼？

因為那顆蛋，不是功能。

是參與感。

也是一種「這是我做的」的感覺。

這種東西如果你只看理性效率，根本看不到。

### 4. 抵觸

這個我覺得最狠。

因為人一旦感覺自己被強迫，

就算那件事本來對他有利，他都會想反著來。

安全帶剛推的時候，有人會拿刀把它割掉。

不是因為他真的覺得安全帶不好。

而是他不喜歡被規定。

我覺得這件事不只在政策裡會發生。

在家庭、團隊、產品導入、教育裡都會發生。

你只要讓人覺得「你都幫我決定好了」，

抵觸就很容易出來。

​▋ ​我自己讀完最有感的是：很多問題不是產品問題，是阻力沒被看見

這句話我覺得很值錢。

因為它會直接改變你怎麼看卡住這件事。

很多時候我們看到一個東西推不動，

第一反應是：

是不是不夠好？
是不是不夠便宜？
是不是不夠吸引人？
是不是講得不夠清楚？

這些當然都可能是原因。

但我覺得真正該補的問題是，

你還要問另外一個問題：

**他到底卡在哪裡動不了？**

這個角度很重要。

因為一旦問題不是「喜不喜歡」，

而是「哪裡卡住」，

你就不會一直往上堆說服。

你會開始往下拆摩擦。

這兩種做法完全不一樣。

前者是一直加力。
後者是先把阻力拿掉。

​▋ ​這件事不只用在賣東西，拿來看工作流也很好用

我自己讀到這裡時，

其實一直在想到工作流。

因為很多流程接不起來，

很多時候也不是因為工具不夠強。

而是阻力沒被處理。

像是：

1. 這套東西要重新學，大家不想改
2. 這段流程要多一個步驟，大家嫌麻煩
3. 這個新做法會讓某些人感覺自己原本那套被否定
4. 這件事是上面直接壓下來的，下面自然抵觸

你看，這完全就是那四種阻力。

所以我覺得這個角度值錢的地方，

不只是拿來做銷售。

它其實很適合拿來看：

- 為什麼專案推不動
- 為什麼新流程接不起來
- 為什麼團隊嘴上說好，最後還是沒動

很多時候不是大家不同意。

而是那個阻力，你根本還沒看到。

​▋ ​我最後留給自己的，是一個很簡單的檢查表

以後只要我覺得一件事推不動，

我會先不急著問：

「是不是這東西不夠好？」

我會先問這四個問題：

1. 他是卡在不熟嗎？
2. 他是覺得太麻煩嗎？
3. 他是不是情感上不舒服？
4. 他是不是覺得自己被強迫了？

這四題，我覺得很有用。

因為它不只幫你看別人，

也幫你看你自己。

很多時候我們自己不想做一件事，

也不是真的因為那件事沒價值。

只是某種阻力剛好卡在那裡。

你如果不把那個卡點拆開，

很容易就只剩下一句：

我最近沒動力。

但其實不是沒動力。

是有阻力。

而且那個阻力，通常是有名字的。

​▋ ​我的結論

**很多事情推不動，不是吸引力不夠，而是阻力太大。**

所以與其想怎麼把話講得更漂亮，

不如先去找：

到底是哪一段讓人不舒服、
哪一段讓人嫌麻煩、
哪一段讓人覺得自己被推著走。

你把那一段拆掉，

很多事情才真的會開始動。
