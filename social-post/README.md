# social-post

一個從原始 `social-post` skill 延伸出來、加入 Fish 個人內容哲學的公開版本。

這個 repo 目前同時包含：

- `SKILL.md`
  - 主 skill 定義與使用規則
- `thinking_profile.md`
  - Fish 發文時的底層思維與內容哲學
- `post_filters.md`
  - 生成貼文前的內容過濾器
- `style_profile.md`
  - 當前整理出的個人語氣檔
- `content_plan.md`
  - 14 天內容日曆
- `references/`
  - skill 依賴的參考檔與公式檔
- `drafts/`
  - 實際產出的貼文草稿
- `rewrites/`
  - 用這套方法改寫過的文章樣本

## 這個版本多了什麼

原始 `social-post` 比較重：

- 平台節奏
- 爆文公式
- CTA
- 發佈與追蹤流程

這個版本額外補進：

- 不是只學語氣，也學內容判準
- 不只像 Fish 在講話，也更像 Fish 在想事情
- 每篇文不只追求順和像，還要能留下新觀點、新判準或可用作法

## Fish 版本的核心發文哲學

1. 真正有價值的是中間那段，不只是結果
2. 不完美比假的完整更有價值
3. 真誠是唯一的語言
4. 要讓人帶走可用的東西，不是只被情緒打到

## 使用方式

如果你只是想看 skill 本體，先讀：

1. `SKILL.md`
2. `thinking_profile.md`
3. `post_filters.md`

如果你想看實戰產出，讀：

- `drafts/`
- `rewrites/`

## 重要規則

- 改寫外部文章 / 筆記 / voicenote 時，不直接覆寫原文
- 改寫版本另存，保留來源原文不動
