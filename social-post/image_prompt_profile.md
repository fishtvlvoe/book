# Image Prompt Profile — Fish / Company CI

> 這份檔案用來生成貼文配圖、封面圖、書摘圖的提示詞。目的不是只做「好看圖」，而是做出一眼就能看出是 Fish / 該品牌會用的圖。

_最後更新：2026-06-14_

## 使用原則

- 先讀文案，再決定圖片
- 圖不是獨立作品，是文案的延伸
- 若有公司 CI，優先服從公司 CI
- 若沒有公司 CI，退回 Fish 個人預設視覺語言

## Fish 個人預設視覺語言

### 關鍵感受

- 安靜
- 克制
- 真實
- 有思考感
- 不是躁動型商業海報

### 常見畫面元素

- 書桌
- 筆記本
- 咖啡杯
- 筆電
- 柔和自然光
- 有留白的工作場景
- 一點點「正在思考」而不是「正在賣」

### 構圖方向

- 左文右物，或右文左物
- 文字區要乾淨，方便上標題
- 不要滿版塞滿元素
- 單一主物件明確，不要同時太多主角

### 視覺風格

- 偏寫實
- 偏 editorial / premium / thoughtful
- 不要過度科技感霓虹
- 不要廉價 startup hero 圖感
- 不要 AI 味太重的假光影、假手、假材質

### 色彩方向

- 中性、木質、奶油白、灰、墨色
- 低飽和
- 需要強調時再放一個主色

## Prompt 輸出格式

每次產圖前，先整理成 4 段：

1. **用途**
   - 這張圖要放哪裡：FB 貼文 / Threads 配圖 / 書摘封面 / 輪播首圖
2. **文案核心**
   - 這篇文最重要的一句話是什麼
3. **視覺翻譯**
   - 把文案翻成畫面，不要只抄文字
4. **風格限制**
   - 要保留哪些品牌語言，要避開哪些常見 AI 圖問題

## 最後檢查

在交出 prompt 前，先問：

1. 這張圖有沒有真的在幫文案說話？
2. 這張圖有沒有套到 Fish / 品牌的視覺語言？
3. 這張圖是不是一眼看起來太像 generic AI 海報？
